PASTA PARA PDFs VIA FTP
=======================

Esta pasta é para PDFs enviados via FTP (sem limite de tamanho).

Como usar:
1. Envie o seu PDF via cliente FTP (FileZilla, Cyberduck, etc.) para esta pasta
2. Pode usar o nome original do ficheiro (ex: catalogo-produtos-2024.pdf)
3. No backoffice, vá a "Categorias" > "Adicionar Card"
4. Selecione "Selecionar do Servidor (FTP)"
5. Clique em "Ver PDFs no Servidor"
6. Selecione o PDF que enviou

Nota: Esta pasta é separada da pasta "pdfs" que contém ficheiros com nomes gerados automaticamente pelo sistema.
